第一次银行家培训

一、印象最深刻的点

0.为什么要做银行家？利他者利己

做银行家，不是为了别人，是为了成就自己，为了让自己实现财务自由、心灵富足。

目标是为了成就自己，过程是带领游戏、点评玩家，通过带游戏，让游戏有序进行，处理好游戏过程中的冲突等问题，让玩家有收获，这样才能实现自己有收获。利他者利己。

现金流游戏参与者的四个level：

  1. 新玩家：10次以下，关注的是术层面，关注自己如何出圈
  2. 老玩家：10次以上，关注市场，关注谈判，关注市场上的机会，关注游戏过程中的情绪
  3. 带领银行家：从单纯的投资技巧到投资系统
  4. 点评银行家：需要找到点评的点，从玩家的每一个move盲点中，从玩家的分享话语中找到点评的要点

马丁老师说，学习是两个神经元之间的连接。 带领银行家与玩家的不同在于，同样的时间里，可以观察、思考和判断每个玩家的决策，建立神经元连接的机会就多了；点评银行家与带领银行家的区别在于，点评银行家除了要观察，还要讲出来，还要想着让玩家有收获。

点评是一个主动学习的过程，点评也有三个阶段，一开始记录每个玩家分别1个点，进行点评，知道自己自己要点评什么内容，表达清楚；然后是说清楚之后，对方要能听的明白；最后是能够引起对方的共鸣，让对方有心情继续听，有启发。

马丁老师建议，做带领银行家3、5次之后，可以加大难度尝试边带领、边参与游戏。

1. 一元股背后的投资系统 - 到底应该买多少？

一元股应该说是公认的好投资，是游戏中获取第一桶金最快捷的一个投资产品，但是到底应该买多少才合适呢？买少了错过了一次巨大的机会，买多了有可能后面面临破产的风险。这背后蕴含了一个专业稳健投资系统的考量逻辑，理解如下。

以一个飞行员为例（总收入9500，总支出6900，月现金流2600，每个小孩支出480，储蓄400），可以从两个角度考虑。

基础数据：按照经验，一元股的市场价格啊1040之间，所以投资回报预期一般是20倍； 游戏中平均的投资收益率在30%40%，我们选30%作为计算参数； 投资收益率 = 月现金流*12/首付资金

1.1 以终为始，为了实现财务自由，我们该买多少股

为了实现财务自由的目标，也就是我们要达到6900的非工作收入，我们需要初始资金=6900*12/30%=276000，那么按照20倍的收益来算，我们需要买1.38万股一元股。

1.2 兼顾考虑最大风险和最大可投入

第一轮的时候，手上最大可投入的资金为一个月的现金流+储蓄+可贷款额度（2600+400+2600*10）= 29000 ，也就是说我们最大可以买2.9万股，那么我们是否要买这么多呢？还要考虑可能存在的风险。

游戏中风险有三种：

  1. 失业风险：需要支出一个月的总支出6900元（可以换算为月现金流690元）
  2. 生孩子的风险：每个月负现金流480
  3. 最大额的支出：游艇首付1000 + 维护费每个月340 = 440/月

一轮下来，最可怕的就是，碰到3次额外支出，然后再遇到一个生孩子的风险，最后再失业，这样要付出的总支出应该就要有6900+480+（1000+340）*3 = 11400.

这样的话，如果只考虑一轮最大程度的风险，那么就是（可投资资源 - 风险储备）= 29000 - 11400 = 17600 ，我们可购买17600 。

综合来说，可购买的数量应该是1.38万到1.76万，这样即能抓住机会实现财务目标，又可以抵御较强大的风险。

1.3 总结

没想到一元股投资的背后居然蕴含了一个基本的投资系统，隐含了这个人对风险偏好、对收益和对目标的考量，参加培训前有比较模糊的概念，重新梳理之后应该说是比较清晰了。

  1. 『风险偏好』：你的目标是什么？你的风险偏好是怎么样的？如果是超高风险偏好的人，可能就直接买2.9万股了，而稍微稳健的做法，可能就是投资1.38万股实现财务自由即可（我记得业胜主席就是报的1.38万这个数字）
  2. 量化，量化，量化，唯有量化才能抵御恐惧，才能有始有终
  3. 游戏中这样的计算，应该在游戏开始前，拿到卡片就要计算好的

1.4 与现实的结合和行动

以终为始，先胜而后战

我的财务自由目标是什么？我需要多少的第一桶金？ 我有多少能够使用的资源？储蓄、每个月的现金流、信用贷、公积金、投资资格（房票）、贷款资格（贷票）等等 我有哪些风险？意外身故、重疾、加息导致的月供陡增等等 我的风险偏好是怎么样的？我愿意承担哪些风险？如何防范、转移这些风险？ 我的房产投资收益率，一般能到多少？

量化上面的每一个问题，由此打造出自己的投资系统来，这个是可以立即着手去做的。

2. 关于财务自由的标准和定义

罗伯特·清崎认为的财务自由，是实现非工资收入超过总支出，不再为金钱工作；

哈佛·艾克与罗伯特·G·艾伦提出的财务自由，是要实现通过多途径的、弱相关性甚至相反的非工资收入，不仅仅要超过（理想生活所需要的）总支出，应该是超过两到三倍的总支出。

2.1 找到人生意义，找到奋斗的动力所在

马丁老师提出一个问题，财务自由就是不再为金钱工作，就是可以做自己想做的事情、从事自己喜欢的事情，那么如果你就在做自己喜欢的事情，赚到的钱能够超出开支&满足需求，那么是否也是一种财务自由呢？

举例说道钱伟长，一开始学习文学和历史，后来为了救国转学物理，终成大家。 马丁老师说，兴趣是最不牢靠的，强调找到人生意义，找到奋斗的动力所在。

我的理解是，财务自由首先是一种自由，要的是被动收入，如果你喜欢的事情仍然是你的主动收入，也就是不给你钱，这件事情你能坚持多久呢？实现财务自由的人，我想是可以一直坚持下去的，而且可以有更多选择的自由。

2.2 游戏中的财务自由

游戏中的财务自由分成三层： 1. 卡车司机：活下去的财务自由，吃馒头住桥洞 2. 经理、工程师：小康财富自由，三口之家，1到2万 3. 医生、飞机驾驶员：奢侈的财务自由，顶层生活

分层是为了明确目标，银行家王若3年时间实现财务自由，罗伯特清崎9岁接受财商学习，47岁实现财务自由。

结果很重要，但更长的时间里，我们是处于过程中，要学会享受过程。

3. 协助而不是帮助

帮助貌似好意，但其实是在限制别人的选择，同时在姿态上有一种我高你低、我主你客、我觉得你不够好需要帮助的感觉，容易引起对方排斥、抗拒的反应。

协助是给对方一个多一种选择，是以对方为主导，对方可以选择，不选择也没有关系。这样我们自己也会比较舒适。

4. 银行家自我介绍

银行家在游戏开头做自我介绍，自我介绍的目的有四个： 1. 吸引别人的兴趣 2. 做出示范，起带头作用（影响他人） 3. 建立联系（老乡？同行业？） 4. 证明资质（为什么要跟你来玩这个游戏？）

证明资质这部分，要挑选有利于自己的，但是不要虚假包装，虚假的东西没有力量。 找一个点就可以了，有如下可以参考： - 工作（央企？500强负责人？） - 过往成绩（通过包租实现29万月现金流？当月买了8套重庆？这次大神太多） - 参加过多少游戏

我自己整理了一个自己的，有点虚哈哈： > 我是潜光，来自福建省福州市，目前在一家央企做互联网保险项目，16年开始玩这个游戏的，通过这个游戏认识了很多大神，自己的财商也得到飞速提升，去年通过房产方面的努力实现千万级别的融资计划。今天由我来带大家一起玩这个游戏，希望大家都能够有所收获，一起进步。

4. 如何收获更大

  • 写总结，不要照搬只是，结合生活体验来总结整理，并最终能够促进行动（心灵创富三段论：印象最深刻的点、与现实的联系、启发和收获）
  • 教给别人，协助别人进步（我把这部分内容整理出来，也算是教人的一个过程吧）
  • 重复训练，多参加复训
  • 勤实践，创造机会实践

5. 其他要点

  • 中间找了一些当天参加游戏的人来，其中有一个人说的一句话让我眼前一亮：“买小生意，竞争的人更多”。这个思路有点类似欧神对大户型的情有独钟，是啊，如果只拿小生意，跟大家就在一个起跑线上，基本上就是看谁的运气好，而买了大买卖，如果能够场上有有钱人的话，可以马上拿到钱。
  • 所有的投资，在付首付的时候、在买入的时候就已经知道是否会赚，先胜而后战，先赚而后买就是这个意思
  • 财商=10%基础知识+90%的情商；情商=觉知情绪+管理情绪；

二、与现实的联系

  1. 量化投资要素（风险），选择一个合适自己的投资系统

去年开始接触房产、接触欧神等大神，自己也有操作，但是内心一直在纠结甚至恐惧的是，负债率这么高会不会有危险？如果经济危机怎么办？

上完这个培训，对投资系统有了更深刻的理解，每个人有自己的风险偏好，一个稳健的理性的投资系统，应该考虑各方面的风险因素，结合自己的财务自由目标进行设定。

  1. 协助和帮助的区别，听完豁然开朗，我有个“不成器”的妹妹，家里人试过各种方式的引导、劝告，一直都没有什么特别的效果，反而让她一直一直有反感、排斥。是时候改变方式了，给他更多的选择，而不是告诉他应该怎么做，是平等的协助而不是我高他低的帮助。

  2. 房产投资的时候，是在买入的时候就已经赚到了，而不是在卖出的时候才知道是不是赚的，也不是一定要放大水才能赚到。重点是在事前的调研，看房200套、了解地段规划政策、了解户型租售比。。。

三、行动和改变

  1. 最好的学习是教给别人，回家就和老婆分享了投资系统的内容，然后各自用飞机驾驶员的职业卡做了计算，居然得出的结论还不一样。然后做了分工，我盘点一下需要再买入多少保险产品，她计算一下现在可用的资源有多少，拿起量化的武器，停止无谓的焦虑。（还有就是拖拖拉拉10天才写完培训总结）

  2. 时刻记住是要协助而不是帮助。开始在朋友圈分享一些文章，单独给老妹和老爸看，想通旁击侧敲的方式影响他们，慢慢来，第一个目标是通过自己发在朋友圈的思考总结，引起老妹的思考或者好奇。

  3. 往后五年看，我希望花多少年实现财务自由，我希望实现什么样的财务自由？有没有可能量化出来，定个小目标？
